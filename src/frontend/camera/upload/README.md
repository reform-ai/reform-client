# Video Upload Component

Handles video file upload for the test/upload feature (App.test.js flow).

## Purpose
- Allows users to upload video files
- Processes uploaded video and extracts frames
- Sends video/frames to backend for analysis

## Flow
1. User selects exercise type
2. User uploads video file
3. Video is processed and sent to backend
4. Receives feedback from backend

## Difference from Live Streaming
- **Live Streaming** (`frontend/camera/`): Real-time frame capture and streaming
- **Video Upload** (`frontend/upload/`): File-based upload and processing
